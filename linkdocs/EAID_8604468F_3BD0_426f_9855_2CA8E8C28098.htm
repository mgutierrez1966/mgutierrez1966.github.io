PROCESOS Y ELEMENTOS FUNCIONALES

 
TIPO
NOMBRE
 DESCRIPCION DEL PROCESO A EJECUTAR
Clic
Modificar
Muestra  la vista FormExpedienteAdquisicion con los datos del expediente. Si la edición del elemento es exitosa, actualiza el modelo.
Clic
Eliminar
 Solicita confirmación para eliminar permanentemente el expediente del sistema, y todos los recursos asociados, en estado "Alta provisional".
Elimina el modelo.
Clic
Alta individual de recurso HW
 Muestra la vista "FormNuevoRecursoHw", pasando los valores por defecto aplicables al nuevo recurso: UCO a cargo,  Fabricante y Distribuidor.
Si la vista tiene exito, asigna la entidad RecursoHw recien creada al expediente, y guarda el modelo. Actualiza el expediente y la vista de recursos asociados.
Clic
Alta masiva de recursos desde fichero
 ** INICIO ARIGIS-113
Muestra la vista "FormCargaMasivaExpAdquisicion", pasando el expediente de adquisición.
Si la vista tiene exito, se actualiza la lista de los recursos del expediente.
** FIN ARIGIS-113
Respuesta de servicio
CargaRecursosExpAdquisicion
 Si la respuesta es correcta, se actualiza la relación de recursos HW asociados al expediente.
Si la respuesta es erronea, se informa al usuario del error producido.
Clic
Finalizar expediente
 Solicita confirmación para finalizar el expediente, informando que los recursos HW del expediente pasarán a estado "Alta", y se finalizará el expediente.
Si se confirma la operación, llama al servicio API "FinalizarExpedienteAdquisicionHw".
Clic
Exportar lista
 Llama a la operacion READ del servicio REST de RecursoHw, con la opcion "csv", filtrando por el expediente de adquisición.
Muestra un mensaje emergente mientras se realiza la operación asincrona.
Cuando el servicio responde, muestra un mensaje con el enlace al fichero generado.
Respuesta de servicio
FinalizarExpedienteAdquisicionHw
 Si la respuesta es correcta, se inicializa la vista para mostrar el expediente modificado.
Si la respuesta es erronea, se muestra el error devuelto por el servicio.
Clic
Ver recurso
 ** INICIO ARIGIS-270
Abre en una nueva pestaña la vista PnRecursoHWDetalle con los datos del recurso seleccionado. La nueva pestaña puede cerrarse.
** FIN ARIGIS-270
