<b>- Precondiciones:</b><br/>- Usuario identificado<br/>- Acceso al tablero como propietario, visor (compartido), colaborador, responsable de tarjeta o informado.<br/><br/><b>Escenario principal:</b><br/>- Un usuario accede al tablero a través de la vista Tableros, o desde un enlace directo.<br/>- El sistema muestra la vista "Tablero" con la siguiente información:<br/> - El nombre del tablero<br/> - Las listas de tarjetas en modo columna, mostrando:<br/> - Nombre de la lista<br/> - Marca de seguimiento (en su caso)<br/> - Las tarjetas en modo miniatura, mostrando:<br/> - Nombre<br/> - Responsable<br/> - Número de colaboradores<br/> - Barra de color según la lista<br/> - Fecha de vencimiento: Se mostrará con diferente color de fondo según la fecha actual y la lista en la que se encuentre:<br/> - Faltan más de 2 días (48 horas): Sin fondo<br/> - Faltan ménos de 2 días: Fondo naranja<br/> - Se ha sobrepasado: Fondo rojo<br/> - Si la tarea se encuentra en una lista de tipo "Finalizado", se usará fondo verde en cualquier caso.<br/><br/>- El usuario puede abrir una tarjeta pulsando sobre su miniatura (ver CU.054: Visualizar datos de una tarjeta)<br/>- El usuario puede mover una tarjeta dentro de su propia lista, cambiando el orden relativo (ver CU.031: Mover una tarjeta a una lista)<br/>- El usuario puede mover una tarjeta a otra lista (ver CU.031: Mover una tarjeta a una lista)<br/>- El usuario puede modificar los datos básicos del tablero, si es propietario del mismo (CU.007: Modificar los datos básicos de un tablero)<br/>- El usuario puede compartir el tablero con otros grupos, si es propietario del mismo (CU.002: Compartir un tablero )<br/>- El usuario puede crear una nueva tarjeta, si es el propietario del tablero (CU.026: Crear una tarjeta )<br/>- El usuario puede añadir una lista al tablero, si es el propietario del tablero (CU.011: Crear una lista)<br/>- El usuario puede guardar el tablero como plantilla, si es el propietario (CU.006: Guardar un tablero como plantilla)<br/>- El usuario puede archivar el tablero, si es el propietario (CU.001: Archivar un tablero)<br/>- El usuario puede eliminar el tablero, si es el propietario (CU.005: Eliminar un tablero)<br/><br/><br/><br/><b>Postcondiciones:</b><br/>N/A<br/><br/>
