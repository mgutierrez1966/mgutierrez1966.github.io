<b>Precondiciones:</b><br/>- Usuario identificado<br/>- El usuario es el responsable de la tarjeta<br/><br/><b>Escenario principal:</b><br/>- Un usuario selecciona la opción "Colaboradores..." del menú de la tarjeta<br/>- El sistema muestra los colaboradores actuales.<br/>- El usuario selecciona un grupo o un miembro, y selecciona "Añadir colaborador"<br/>- El sistema añade al miembro o grupo como colaborador de la tarjeta.<br/>- El usuario selecciona un miembro o grupo de la lista de colaboradores y selecciona "Quitar como colaborador"<br/>- El sistema elimina el miembro o grupo de la lista de colaboradores<br/>- El usuario confirma la operación<br/><br/><b>Postcondiciones:</b><br/>- El sistema ha enviado notificaciones del tipo "COLABORADOR_TARJETA" al miembro o grupo añadido como colaborador.<br/>- El sistema ha enviado notificaciones del tipo "NO_COLABORADOR_TARJETA" al miembro o grupo eliminado como colaborador.<br/>
