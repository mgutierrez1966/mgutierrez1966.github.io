Arbol de Orgánica ET
Arbol
 Permite navegar por la estructura orgánica del ET, y seleccionar un nodo para mostrar los datos de detalle.
Muestra el arbol de Estructura Orgánica del ET

Para cargar el primer nivel, se usa el servicio API LeerNodosSubordinadosOrganicaEt, sin especificar un nodo raiz (de esa forma devolverá los nodos de primer nivel del ET)

Cuando se depliegue un nodo del arbol, se cargarán los nodos subordinados, llamando al servicio API LeerNodosSubordinadosOrganicaEt, pasando como parámetro (nodo_raiz) el identificador del nodo que se quiere deplegar.

Inicialmente el arbol mostrará unicamente un nivel desplegado.

El arbol permite la funcionalidad "arrastrar y soltar" entre nodos, si el rol de usuario es ADMINISTRADOR.
Los nodos del arbol, se pueden mover con las siguientes condiciones:
  • Un nodo no se puede mover a un nodo subordinado
  • Un nodo no se puede mover a su mismo nodo padre
  • Un nodo de tipo MandoEt no se puede mover bajo un nodo de tipo DependenciaEt o Almacén
** INICIO ARIGIS-669
  • Un nodo de tipo CECOM no puede moverse bajo un nodo que NO sea de tipo BT
** FIN ARIGIS-669
Detalles de UCO
Panel
 Asociado a la entidad seleccionada en el árbol de orgánica ET.

Tiene una disposición de campos diferente según el tipo de nodo (ver boceto)

En la parte superior del panel se muestra la ruta completa del nodo, en forma de "bread crumbs" o migas de pan.
Usuarios asociados
Grid
 Tabla con la lista de usuarios asociados al nodo de la orgánica seleccionado.
Su contenido proviene de diferentes servicios API en función del tipo de nodo seleccionado:

- Mando Orgánico: servicio API  LeerUsuariosMandoOrganico
- DependenciaEt: servicio API LeerCispocPorUco
- Almacen: servicio API LeerUsuariosAlmacen
- BT: servicios API
      - Pestaña "Usuarios SUBDIRECTOR": LeerUsuariosSubdirector
      - Pestaña "Usuarios CISPOC": LeerCispocPorUco
- CECOM: servicio API
      - Pestaña "Usuarios OACISE": LeerUsuariosOacise
      - Pestaña "Usuarios CISPOC": LeerCispocPorUco

En todos los casos se pasa al servicio API el valor del identificador del nodo seleccionado.

La columna "Usuario" muestra el atributo "uid" del usuario.

En la primera columna se muestra un botón para editar el usuario correspondiente. Este opción solo se muestra para el rol ADMINISTRADOR.

Arrastrar y soltar
Nodo de arbol de orgánica
 Si la acción de arrastrar y soltar está permitida (ver condiciones), se modifica el atributo "nodoSuperior" del nodo arrastrado, y se llama al servicio REST de la entidad MandoEt, DependenciaEt, Almacen, Bt o Cecom  (según el tipo de nodo) con la operacion UPDATE sobre el nodo modificado.
Si la operación se ejecuta correctamente, se mantiene la acción del usuario. Si la operación da un fallo, se revierte el nodo a su posición original.

Botón "Exportar nivel UCO"
 ** INICIO ARIGIS-667
Se solicita confirmación con el texto "Se realizará una exportación a CSV de la estructura orgánica hasta nivel UCO con CIU. ¿Desea continuar?"

Se hace una llamada al servicio REST de la entidad DependenciaEt, usando el path "/csv" para la obtención de la exportación a CSV, que devuelve un enlace al fichero CSV generado con la exportación.

Como primer campo de la exportación se indica el atributo "pathCsv", y adicionalmente se incluyen los datos básicos de la entidad DependenciaEt: CIU, nombreCorto, nombreLargo, dirección, cp, teléfono, email.
Si la operación es correcta, se muestra en un mensaje "toast" el enlace del fichero a descargar.
