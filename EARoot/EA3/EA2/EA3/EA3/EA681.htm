Appears In: Gestión de licencias software
El usuario SEAPO o CISPOC pueden acceder a la visualización de los detalles de una licencia SW, desde la vista de Consulta de licencias SW.<br/><br/>El sistema muestra los siguientes datos de la licencia:<br/><br/>- Datos básicos:<br/> * Fecha de adquisición<br/> * Estado (alta / descatalogado / baja)<br/> * Ámbito de uso (individual / red)<br/> * Duración (perpetua / suscripción)<br/> * Fechas de inicio y fin de validez (para suscripciones)<br/> * Fecha de descatalogación<br/> * Fecha de baja<br/> * Código de licencia<br/> * Versión instalada<br/> * Versión máxima soportada (por el contrato de mantenimiento en vigor, si lo hay)<br/> * Número de usuarios permitido (para licencias con ámbito red)<br/> * Fabricante (según tabla de empresas gestionada por SEAPO)<br/> * Distribuidor (ídem)<br/> * Soporte (ídem)<br/> * UCO a cargo<br/> * POC relacionado con el uso de la licencia<br/><br/>- Datos del Expediente de adquisición SW (código SEAPO)<br/>- Producto software (según catálogo de productos SW que mantendrá SEAPO, de forma similar al catálogo de HW)<br/>- Expedientes de mantenimiento que afectan a la licencia<br/>- Histórico: <br/> * Datos de alta<br/> * Reasignaciones entre UCO<br/> * Actualizaciones de versión<br/> * Cambios de estado (descatalogada y baja)<br/><br/>
