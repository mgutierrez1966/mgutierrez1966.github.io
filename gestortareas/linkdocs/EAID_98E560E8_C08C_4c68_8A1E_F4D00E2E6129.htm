PROCESOS Y ELEMENTOS FUNCIONALES

 
TIPO
NOMBRE
 DESCRIPCION DEL PROCESO A EJECUTAR
Clic
Autoridades informadas - Añadir
 Muestra la vista FormSeleccionParticipanteMultiple, con las opciones:
- permitirPersonas: true
- permitirGrupos: true

Al recuperar el control, llama a la operacion CREATE del REST de InformadoTablero, para cada uno de los particpantes seleccionados que no estén ya en la lista actual.
Cuando responsa el servicio, actualiza la lista con los nuevos participantes.
Clic
Compartidos - Añadir
 Muestra la vista FormSeleccionParticipanteMultiple, con las opciones:
- permitirPersonas: false
- permitirGrupos: true

Al recuperar el control, llama a la operacion CREATE del REST de CompartidoGrupo, para cada uno de los grupos seleccionados que no estén ya en la lista actual.
Cuando responsa el servicio, actualiza la lista con los nuevos grupos.
Clic
Icono X de un participante
 Solicita confirmación de la operación de borrado.
Llama a la operacion DELETE del REST de InformadoTablero o CompartidoGrupo, según el grupo de participantes.
Cuando responsa el servicio, actualiza la lista con los participantes restantes.
Cerrar vista
Al cerrar la vista, se devolverá el número total de interesados, como suma de autoridades informadas y grupos compartidos.
