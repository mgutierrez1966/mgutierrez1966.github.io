PnRecursoHWDetalle
VERSION
4 (ARIGIS-738)
OBSERVACIONES
Muestra la vista de detalle de un recurso informático HW.
DATOS ENTRADA
  • recursoHw: Obligatorio. Indica el objeto o el ID del recursoHw a mostrar en la vista.
** INICIO ARI-17
  • permiteEdicion: Opcional. Indica si se permiten operaciones de modificación. Si es falso, no se muestran las opciones relacionadas con la modificación del recurso. Por defecto "Cierto".

** FIN ARI-17

Pestaña "Datos del recurso"
TabPanel
 Muestra los datos del recurso en forma de solo lectura
** INICIO ARIGIS-261
En el campo "Tipo de ubicación", si el valor es "En tránsito", se muestra además un enlace a la reasignación en la que está incluido el recurso. La reasignación la obtiene de la API "InventarioHwAPI/LeerReasignacionEnTransitoPorRecurso", pasando el ID del recurso. Esta API devuelve el valor nulo si el recurso no está en tránsito, o el objeto ReasignacionHw asociado en caso contrario.

PROCESOS Y ELEMENTOS FUNCIONALES

 
TIPO
NOMBRE
 DESCRIPCION DEL PROCESO A EJECUTAR
Clic
Modificar Recurso
 ** INICIO ARI-24
Se muestra la vista "FormModificarRecursoHw" para modificar los datos del recurso HW.

Si el resultado de la vista es correcto, se actualizan los datos en el recurso. Se actualiza la vista del dato en el panel de "Datos del recurso".
** FIN ARI-24
 
Clic
Generar etiqueta
 ** INICIO ARIGIS-33
Llama al servicio API asíncrono InventarioHwAPI.generarInformeEtiquetas(), pasando como parámetro de filtrado el ID del recurso HW.
Si el servicio responde con éxito, muestra el enlace de descarga del fichero devuelto por el servicio en un mensaje emergente (toast).
** FIN ARIGIS-33
Clic
Crear solicitud correctivo
 ** INICIO ARIGIS-673
- Se llama a la API MantenimientoHwAPI.obtenerContratoMantenimientoHwEnVigor(), para obtener el objeto ContratoMantenimiento en vigor actualmente. Si la llamada devuelve un objeto nulo, se muestra un error al usuario informando de que "No se puede crear la solicitud porque no existe ningún contrato de mantenimiento de HW en vigor actualmente".

Si la llamada es correcta, se muestra la vista FormSolicitudMtoCorrecHW, pasando como parámetro "recurso" el objeto RecursoHw actual.
Si la vista finaliza correctamente, se muestra al usuario un mensaje informando de que se ha creado la solicitud de mantenimiento, con el número "XXXXX".
** FIN ARIGIS-673
 
Clic
Sustitución por garantía
 ** INICIO ARIGIS-738
Se muestra la vista "FormGarantiaRecursoHw" para modificar los datos del recurso HW tras la aplicación de la garantía.

Si el resultado de la vista es correcto, se actualizan los datos en el recurso. Se actualiza la vista del dato en el panel de "Datos del recurso".
** FIN ARIGIS-738
 
Clic
Añadir anotación
 Muestra la vista FormAnotacionHistoricoRecursoHw, para editar los datos de un nuevo evento histórico de tipo "Anotacion".
Si la vista se ejecuta con éxito, refresca la lista de eventos históricos del recurso.
 
Clic
Editar anotación
 Muestra la vista FormAnotacionHistoricoRecursoHw, para editar los datos del evento seleccionado.
Si la vista se ejecuta con éxito, refresca la lista de eventos históricos del recurso.
Clic
Borrar anotación
 Solicita confirmación para eliminar el evento seleccionado.
Llama a la operación DELETE del servicio REST de HistoricoHw.
Clic
Enlace "Ver reasignación"
 ** INICIO ARIGIS-261
Abre una nueva pestaña con la vista "PnReasignacionHWDetalle", pasando como entrada el identificador de la reasignación asociada al enlace.
** FIN ARIGIS-261
