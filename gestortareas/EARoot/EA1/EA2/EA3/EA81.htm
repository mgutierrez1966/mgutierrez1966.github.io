<b>Precondiciones:</b><br/>- Usuario identificado<br/>- El usuario es propietario del tablero donde se ubica la tarjeta, o es el responsable de la tarea superior.<br/><br/><b>Escenario principal:</b><br/>- Un usuario accede al formulario de la tarea<br/>- El usuario pulsa sobre alguno de los datos básicos:<br/> - Titulo<br/> - Descripción<br/> - Fechas de inicio, de vencimiento o límite de actuación<br/>- El sistema muestra el dato y permite su modificación.<br/>- El usuario modifica el dato<br/>- El sistema actualiza los datos básicos de la tarea.<br/>- El sistema actualiza el formulario con los nuevos datos y la actividad<br/>- El sistema actualiza las vistas asociadas donde se muestra la tarea (tablero o lista de tareas)<br/><br/><br/><b>Postcondiciones:</b><br/>- El sistema ha registrado la actividad en la tarea<br/><br/>
