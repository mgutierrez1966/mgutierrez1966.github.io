Barra de opciones
Toolbar
 Opciones del usuario:
  • Crear UCO: Menú con las siguientes opciones:
          -  Batallón de Transmisiones (BT): Habilitado si:
    • El rol de usuario es ADMINISTRADOR
    • Está seleccionado el nodo raiz (RT-22).
           -  CECOM: Habilitado si:
    • El rol de usuario es ADMINISTRADOR
    • Hay un nodo seleccionado en el árbol
    • El nodo seleccionado en el árbol de la orgánica es de tipo BT.

 
  • Modificar:
    • Visible solo para rol ADMINISTRADOR
    • Habilitado si hay un nodo seleccionado, diferente del nodo raiz.
  • Borrar:
    • Visible solo para rol ADMINISTRADOR
    • Habilitado si hay un nodo seleccionado y dicho nodo no tiene nodos subordinados
Arbol de Estructura de Apoyo
Arbol
 Permite navegar por la estructura de apoyo SICIS, y seleccionar un nodo para mostrar los datos de detalle.
Muestra el nodo RT-22 como nodo raiz. En el segundo nivel muestra los nodos de tipo "BT". De cada nodo "BT" cuelgan los nodos de tipo "CECOM" que dependen orgánicamente.

Para cargar los nodos de tipo BT, se usará el servicio REST de la entidad Bt.
Para cargar los nodos de tipo CECOM de cada BT, se usará el servicio REST de la entidad Cecom, filtrando por el nodo superior.

Inicialmente el árbol mostrará únicamente desplegados los nodos de tipo BT.
 
Pestaña "Datos básicos" - Detalles de BT o CECOM
Panel
 Asociado a la entidad seleccionada en el árbol de orgánica ET.

Tiene una disposición de campos diferente según el tipo de nodo (ver boceto)

En la parte superior del panel se muestra la ruta completa del nodo, en forma de "bread crumbs" o migas de pan.
Pestaña "Datos básicos" - Usuarios asociados
Grid
 Tabla con la lista de usuarios asociados al nodo de la orgánica seleccionado.
Su contenido proviene de diferentes servicios API en función del tipo de nodo seleccionado:

** INICIO ARIGIS-154
- BT: servicios API
      - Pestaña "Usuarios SUBDIRECTOR": LeerUsuariosSubdirector
      - Pestaña "Usuarios CISPOC": LeerCispocPorUco
- CECOM: servicio API
      - Pestaña "Usuarios OACISE": LeerUsuariosOacise
      - Pestaña "Usuarios CISPOC": LeerCispocPorUco
** FIN ARIGIS-154

En todos los casos se pasa al servicio API el valor del identificador del nodo seleccionado.

La columna "Usuario" muestra el atributo "uid" del usuario.

** INICIO ARIGIS-271
En la primera columna se muestra un botón para editar el usuario correspondiente. Este opción solo se muestra para el rol ADMINISTRADOR.
** FIN ARIGIS-271
 
Pestaña "UCO,s. apoyadas"
Tab panel
 Gestiona las UCO,s. (dependencias) apoyadas por el CECOM seleccionado en el arbol.
 
Pestaña "UCO,s. apoyadas" - Barra de opciones
Toolbar
 Solo visible si el rol de usuario es ADMINISTRADOR.

Opciones del usuario:
  • Agregar dependencias apoyadas: Habilitado siempre.
  • Quitar dependencias apoyadas: Habilitado solo si hay un nodo seleccionado en el arbol.
 
Pestaña "UCO,s. apoyadas" - Árbol de UCO,s. apoyadas
Árbol
 Muestra en forma de árbol, los nodos DependenciaEt que son apoyados por el CECOM seleccionado.

Para obtener los nodos de cada nivel del árbol, se llama al servicio API EstructuraApoyoSicisAPI.LeerNivelArbolApoyoOacise(), pasando como parámetro el CECOM seleccionado, y el nodo superior. En el caso de los nodos raiz, se pasa un valor nulo como nodo superior.
 
Pestaña "UCO,s. apoyadas" - Detalle de UCO
Panel
 Muestra los detalles del nodo seleccionado en el árbol de UCO,s. apoyadas.

Este panel se corresponde con la vista de detalle de un nodo de tipo "DependenciaEt" (ver vista PnGestionEstructuraOrganica)

PROCESOS Y ELEMENTOS FUNCIONALES

 
TIPO
NOMBRE
 DESCRIPCION DEL PROCESO A EJECUTAR
Mostrar
Vista
 Se inicializa el nivel de nodos de tipo "BT" del árbol de estructura de apoyo SICIS (ver descripción del elemento)
 
Clic en un nodo o cambio de nodo seleccionado
Nodo del arbol de estructura de apoyo SICIS
 Se actualizan los datos del panel "Detalles de BT o CECOM", mostrando el panel de detalle según el tipo de nodo seleccionado.

Se actualiza el contenido del panel de "Usuarios asociados" según el tipo de nodo seleccionado (ver descripción del elemento).

Para los nodos de tipo CECOM, se cargan los nodos raiz del árbol de UCO,s. apoyadas.

NOTA: Para mejor usabilidad, aunque cambie el nodo seleccionado en el árbol, se mantendrá activada la misma pestaña (Datos básicos o UCO,s. apoyadas).
 
Clic
Botón Crear -> Batallón de Transmisiones (BT)
 Se muestra la vista "FormBt" para crear un nuevo nodo de tipo BT.
Si la vista se ejecuta con éxito, se actualiza el árbol y se selecciona el nodo recién creado.
 
Clic
Botón Crear -> CECOM
 Se muestra la vista "FormCecom" para crear un nuevo nodo de tipo CECOM.
Si la vista se ejecuta con éxito, se actualiza el árbol y se selecciona el nodo recién creado.
 
Clic
Botón "Modificar"
 Se muestra la vista "FormBt" , pasando como parámetro el identificador del nodo seleccionado.
Si la vista se ejecuta correctamente, se actualizan los datos del nodo en el árbol y en la vista de detalle.
Clic
Botón "Borrar"
 Se solicita confirmación para eliminar el nodo seleccionado.
Se elimina el nodo llamando a la operación DELETE del servicio REST de la entidad Bt o Cecom, en función del tipo de nodo a borrar.
Si la operación es correcta, se actualiza la información del árbol.
Clic
Botón "Agregar dependencias apoyadas"
 Se muestra la vista FormSeleccionUCO, pasando como parámetros:
- filtrarPorUsuario: falso
- nodoRaiz: nulo
- permiteSeleccionarMando: falso
- permiteSeleccionarDependenciaConCIU : cierto
- permiteSeleccionarDependenciaSinCIU : cierto.

Si la vista se ejecuta correctamente, se solicita confirmación al usuario para añadir la dependencia seleccionada y todas sus dependencias subordinadas a la lista de UCO,s. apoyadas por el CECOM.
Se llama al servicio EstructuraApoyoSicisAPI.AgregarNodoCargoApoyadoPorOacise(), pasando el CECOM y el nodo seleccionado.
Si la llamada es correcta, se actualiza la información del árbol.
 
Clic
Botón "Quitar dependencias apoyadas"
 Se solicita confirmación para quitar la dependencia seleccionada y todas las dependencias subordinadas de la lista de UCO,s. apoyadas por el CECOM.
Se llama al servicio EstructuraApoyoSicisAPI.QuitarNodoCargoApoyadoPorOacise(), pasando el CECOM y el nodo seleccionado.
Si la llamada es correcta, se actualiza la información del árbol.
Clic
Botón "Editar usuario" en lista de usuarios
 ** INICIO ARIGIS-271
Muestra la vista FormModificarUsuario, con los datos del usuario correspondiente a la fila.
** FIN ARIGIS-271
