PROCESOS Y ELEMENTOS FUNCIONALES

 
TIPO
NOMBRE
 DESCRIPCION DEL PROCESO A EJECUTAR
Iniciar vista
Carcar la lista de tableros de los que el usuario es propietario, mediante la API TablerosAPI/leerTablerosComoPropietario.
 
Cambio de seleccion
Combobox tablero
 Si hay tablero seleccionado, se carga la lista de listas del tablero, mediante el REST de Lista, filtrando por el ID del tablero. Finalmente se habilita el combobox.

 
Clic
Enviar a tablero
 Se llama a la API TablerosAPI/moverTarjetaALista, pasando la tarjeta y la lista de destino (el parámetro "orden" se deja a nulo, para que se mueva al final de la lista).

Al responder el servicio, se cierra la vista.
