POSTCONDICIONES
#2
 El orden asignado a una nueva lista es:
- Si es de tipo BORRADOR, el orden es 0 (cero)
- Si es de tipo FINALIZADO, el orden es 999
- Si es de tipo NORMAL o LIBRE, se le asigna el orden de la última lista de tipo NORMAL o LIBRE, incrementado en 1. Es decir, las nuevas listas quedan a la derecha del tablero (antes de Finalizado, si existe).
Nº
INVARIANTES
#3
En un tablero solo puede existir, como máximo, una lista de tipo BORRADOR, y una lista de tipo FINALIZADO.
#4
El nombre de una lista debe ser único en cada tablero.

POSTCONDICIONES
#6
 - Si se ha modificado el tipo de lista, el orden debe ser recalculado:
    - Si el nuevo tipo es BORRADOR, el orden es 0 (cero)
    - Si el nuevo tipo es FINALIZADO, el orden es 999
    - Si el nuevo tipo es NORMAL o LIBRE y el anterior era BORRADOR, la lista debe quedar en el orden = 1, reordenando el resto.
    - Si el nuevo tipo es NORMAL o LIBRE y el anterior era FINALIZADO, la lista debe quedar con el orden de la última lista de tipo NORMAL o LIBRE del tablero, incrementado en 1.
