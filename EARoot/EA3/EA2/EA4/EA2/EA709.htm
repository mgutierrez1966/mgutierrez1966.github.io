Desde la <b>vista del Catálogo de Artículos</b>, el usuario puede seleccionar una categoría y subcategoría, y seleccionar la opción "Nuevo artículo". El sistema solicita los datos del nuevo artículo de la <u>subcategoría seleccionada</u>.<br/>Los datos solicitados de un nuevo artículo son:  <br/><ul>
<li>Nombre (*A100)</li><li>Marca y Modelo (*A100): El sistema propondrá valores similares al tecleado por el usuario para evitar duplicidades. Debe ser un valor único en el Catálogo.</li><li>Descripción (A1000)</li><li>Importe libre de impuestos (CUR)</li><li><font color="#0f0f0f">Estado: En vigor (por defecto) / Planificado / Descatalogado</font></li><li>Fabricante (id): Obligatorio si se especifica un código de fabricante.</li><li>Distribuidor (id)</li><li>Soporte Técnico (id)</li></ul>
<br/>Para especificar el fabricante, distribuidor y soporte técnico, el sistema mostrará una vista de empresas dadas de alta.<br/><br/>El valor "Estado" se establecerá inicialmente a "En vigor"<br/><br/>El sistema también permite establecer el valor por defecto para los atributos del artículo (ver CU.SEAPO.HW.023)<br/><br/>El sistema crea un nuevo artículo y le asigna un Número de Catálogo (NCAT) compuesto por:<br/><ul>
<li>Letra "E"</li><li>El prefijo de la subcategoría a la que pertenece</li><li>Un número de 6 cifras único dentro del Catálogo</li></ul><p/>
