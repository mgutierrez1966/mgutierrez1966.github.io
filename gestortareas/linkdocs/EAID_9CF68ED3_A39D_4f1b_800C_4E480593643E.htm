PROCESOS Y ELEMENTOS FUNCIONALES

 
TIPO
NOMBRE
 DESCRIPCION DEL PROCESO A EJECUTAR
Cambio de texto
Buscar
 Actualiza la lista de usuarios aplicando el filtro "query" con el texto introducido.
Si el cuadro de texto no tiene valor, se elimina el filtro.
 
Clic
Agregar usuario
 Muestra la vista FormNuevoUsuario.

Al recuperar el control, actualiza la lista de usuarios, y muestra la vista FormUsuario, pasando el UID del usuario creado.
 
Clic
Quitar usuario
 Solicita confirmación al usuario para realizar la operación. Le informa que el usuario no será eliminado del sistema, pero ya no podrá acceder a la gestión de tareas de la organización.

Llama a la API AdminApi/bajaUsuarioOrganizacion, pasando el id del usuario seleccionado y el id de la organizacion.

Al recuperar el control, actualiza la lista de usuarios.
