Iniciar vista
Recupera los grupos a los que pertenece el usuario. Se obtiene del REST de MiembroGrupo, filtrando por el ID del usuario.

Crea una pestaña para cada grupo al que pertenece el usuario, y activa la primera de ellas.


 
Activar pestaña de grupo
Recupera los tableros visibles al usuario llamando a la API TablerosAPI/leerTablerosComoMiembro, pasando el ID del objeto MiembroGrupo asociado a la pestaña.

Muestra los tableros agrupados por propietario.

La etiqueta de cada tablero es el título. El color de cada tablero es el definido en el atributo color.
Al pasar el cursor por cada miniatura de tablero debe mostrar un cursor de tipo "mano", y resaltar la miniatura.
