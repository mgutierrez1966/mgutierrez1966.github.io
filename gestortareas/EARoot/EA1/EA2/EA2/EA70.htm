<b>Precondiciones:</b><br/>- Usuario identificado <br/>- El usuario es el propietario del tablero<br/>- El usuario es miembro de algun grupo con el que se ha compartido el tablero.<br/>- El usuario es autorizado a alguna de las listas del tablero<br/><br/><b>Escenario principal:</b><br/>- El usuario selecciona la opción "Seguir lista" del menú contextual de la lista<br/>- El sistema solicita confirmación al usuario. Le informa que al seguir la lista se le notificará cada vez que una tarjeta se mueva a dicha lista.<br/>- El sistema añade al usuario como seguidor de la lista<br/> - Si el usuario ya es seguidor de la lista, directamente o a través de alguno de sus grupos, el sistema informa de ello al usuario.<br/>- El sistema añade una marca de seguimiento en la cabecera de la lista<br/><br/><br/><b>Postcondiciones:</b><br/>N/A<br/><br/>
