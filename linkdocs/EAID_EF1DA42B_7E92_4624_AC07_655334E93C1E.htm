ELEMENTOS VISUALES

 
NOMBRE
TIPO
OBSERVACIONES
Titulo del formulario
Para creación: "Nueva solicitud de suministro de material HW"
Para modificación: "Modificación de solicitud de suministro de material HW nº XXXXXXXXX"
Texto de ayuda
Texto explicativo en la cabecera del formulario
Formulario
Formulario para rellenar los datos de la solicitud. Modal.
Campo "Material inventariable"
Checkbox
 ** INICIO ARIGIS-1105
Asociado al atributo "inventariable". Indica si el suministro es de material inventariable o de componentes.
** FIN ARIGIS-1105
Campo "Peso "
Texto
 ** INICIO ARIGIS-1083
Indica el peso de los artículos a suministrar. Tiene un botón "Calcular", para realizar el cálculo del peso en base a las tablas. Si el campo no tiene valor, se muestra la etiqueta "(sin valor)" en color rojo.
** FIN ARIGIS-1083
Campo "Codigo"
Texto
Muestra el código de la solicitud, o "Pendiente de asignar" para nuevas solicitudes.
Campo "Nueva UCO a cargo"
Selector de UCO
Requerido. Selecciona la UCO a la que estará a cargo el material suministrado.
Campo "Ubicación de entrega"
Selector de Dependencia
Requerido. Selecciona la dependencia de la UCO donde se entregará el material a suministrar.
Campos "Dirección", "CP", "Provincia"
Se corresponden con los atributos de la entidad "SolicitudMantenimiento". Cuando se selecciona una ubicación de entrega, se toman los datos de dicha ubicación para estos atributos. No obstante el usuario puede modificarlos a voluntad.
Campo "Contacto (CISPOC)"
Combobox
 Corresponde al campo "pocCis" de la entidad "SolicitudMantenimiento".
Permite tomar los datos (empleo, nombre y apellidos) de la lista de CISPOC de la UCO a cargo.
El valor del campo es libre, no está restringido a los valores de la lista.
Campo "Teléfono"
Teléfono de contacto para la entrega en la UCO. Cuando se selecciona un CISPOC, se toman sus datos para este atributo. No obstante el usuario puede modificarlo a voluntad.
Campo "Comentarios Director"
** INICIO ARIGIS-1136
Solo es modificable por el rol DIRECTOR. Para el resto de roles aparece como solo lectura.
** FIN ARIGIS-1136
Barra de botones
Toolbar
 Botones:
- Guardar: activado solo si el formulario es válido.
- Cancelar

 DESCRIPCION DEL PROCESO A EJECUTAR
Inicializar vista
Para creación:
- Se llama a la API MantenimientoHwAPI.obtenerContratoMantenimientoHwAbierto(), para obtener el objeto ContratoMantenimiento en vigor actualmente. Si la llamada devuelve un objeto nulo, se muestra un error al usuario informando de que "No se puede crear la solicitud porque no existe ningún contrato de mantenimiento de HW en vigor actualmente".

- Crear un objeto SolSuministroMatUco nuevo e inicializar valores por defecto.


Para edición:
- Tomar valores del objeto a editar
- Iniciar edición del modelo
Clic
Calcular (peso)
 ** INICIO ARIGIS-1083
Se muestra la vista FormCalculadoraPesos.
Si la vista se ejecuta con éxito, se actualiza el valor del atributo "peso" de la solicitud con el valor "peso" devuelto por el formulario.
** FIN ARIGIS-1083
Clic
Selector de Nueva UCO a cargo
 Se muestra la vista FormSeleccionUCO,  para la selección de una Dependencia con CIU. Los parámetros de llamada son:
        - permiteSeleccionarMando: falso,
        - permiteSeleccionarDependenciaSinCIU: falso,
        - permiteSeleccionarAlmacen: cierto
 
Cambio de valor
Nueva UCO a cargo
 Se inicializa el campo "Ubicación de entrega" con el mismo valor del nodo de "Nueva UCO a cargo".
Se actualizan los campos "Dirección", "CP" y "Provincia" con los datos correspondientes del nodo seleccionado.
Clic
Selector de Ubicación de entrega
 Se muestra la vista FormSeleccionUCO,  para la selección de una Dependencia de la UCO. Los parámetros de llamada son:
        - nodoRaiz: nodo seleccionado en el campo "Nueva UCO a cargo"
        - permiteSeleccionarMando: falso,
        - permiteSeleccionarAlmacen: cierto
 
Cambio de valor
Ubicación de entrega
Se actualizan los campos "Dirección", "CP" y "Provincia" con los datos correspondientes del nodo seleccionado.
Despliegue de combobox
Contacto (CISPOC)
Los datos de la lista se obtiene del servicio API "leerCispocPorUco", pasando como parámetro el id de la nueva UCO a cargo.
