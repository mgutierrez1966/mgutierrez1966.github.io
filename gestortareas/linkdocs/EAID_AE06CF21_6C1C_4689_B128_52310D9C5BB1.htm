OBSERVACIONES
Color de fondo
Selector de color
 Permite seleccionar un color para el fondo del tablero. Por defecto color blanco.
 
Propietario
Selector de grupo
 Permite seleccionar un grupo de los que pertenece el usuario. Obtiene los datos del servicio REST de Grupo, filtrado por los grupos del usuario (miembroGrupo.miembro.usuario.uid)

Se preseleccionará el primero de la lista.
 
Usar plantilla
Selector de plantilla
 Permite seleccionar una plantilla de tablero existente. Obtiene los datos del servicio REST de Plantilla, filtrando por tipo TABLERO.

Su valor inicial será la plantilla que tenga el atributo "porDefecto" a TRUE. Si no hubiera ninguna, no se establecerá valor inicial.
