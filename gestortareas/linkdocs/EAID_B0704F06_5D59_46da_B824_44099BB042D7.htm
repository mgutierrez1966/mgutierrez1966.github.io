PROCESOS Y ELEMENTOS FUNCIONALES

 
TIPO
NOMBRE
 DESCRIPCION DEL PROCESO A EJECUTAR
Clic
Agregar usuario a grupo
 Muestra la vista FormSeleccionParticipanteUnico, con la opcion "permitePersonas" igual a FALSE.

Si se ha seleccionado un grupo, se muestra la vista FormSeleccionarTipoUsuario, para seleccionar el tipo de usuario en el grupo.

Finalmente se llama a la API AdminAPI/agregarMiembrosGrupo, pasando el ID del miembro asociado al usuario, el ID del grupo seleccionado y el tipo de usuario seleccionado.
 
Clic
Quitar usuario de grupo
 Solicita confirmacion del usuario para realizar la operación.

Llama a la API AdminAPI/quitarMiembroGrupo, pasando el ID de la entidad MiembroGrupo seleccionada en la tabla.
 
Clic
Guardar
 Guarda los datos de la entidad PreferenciasUsuario mediante el REST de esta entidad.
