PROCESOS Y ELEMENTOS FUNCIONALES

 
TIPO
NOMBRE
 DESCRIPCION DEL PROCESO A EJECUTAR
Clic
Modificar
 Muestra la vista FormGrupo, pasando el grupo a editar.

Al recuperar el control actualiza los datos del bloque "Datos básicos".
 
Clic
Agregar usuario a grupo
 Muestra la vista FormSeleccionParticipanteMultiple, con la opcion "permiteGrupos" igual a FALSE.

Si se ha seleccionado algún usuario, se muestra la vista FormSeleccionarTipoUsuario, para seleccionar el tipo de usuario en el grupo.

Finalmente se llama a la API AdminAPI/agregarMiembrosGrupo, pasando los UIDs seleccionados, el ID del grupo seleccionado y el tipo de usuario seleccionado.
 
Clic
Quitar usuario de grupo
 Solicita confirmacion del usuario para realizar la operación.

Llama a la API AdminAPI/quitarMiembroGrupo, pasando el ID de la entidad MiembroGrupo seleccionada en la tabla.
