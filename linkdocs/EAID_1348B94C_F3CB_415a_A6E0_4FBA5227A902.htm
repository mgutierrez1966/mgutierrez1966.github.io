PROCESOS Y ELEMENTOS FUNCIONALES

 
TIPO
NOMBRE
 DESCRIPCION DEL PROCESO A EJECUTAR
Carga de la vista
Se abre el almacén correspondiente a los contratos de mantenimiento. Servicio REST de la entidad ContratoMantenimiento.
 
Clic
Botón Nuevo contrato
 Se muestra la vista FormContratoMantenimientoHw sin ningún parámetro asociado.
Si la vista se ejecuta con éxito, se actualiza el almacén asociado al grid, mostrando el nuevo contrato creado.
Clic
Botón Modificar contrato
 Se muestra la vista FormContratoMantenimientoHw, pasando el contrato seleccionado en el grid como parámetro.
Si la vista se ejecuta con éxito, se actualiza el almacén asociado al grid
Clic
Botón Borrar contrato
 Solicita confirmación para borrar el contrato seleccionado.
Elimina el contrato mediante la operacion DELETE  del servicio REST ContratoMantenimiento.
Si la operación tiene éxito, se actualiza el almacén asociado al grid.
