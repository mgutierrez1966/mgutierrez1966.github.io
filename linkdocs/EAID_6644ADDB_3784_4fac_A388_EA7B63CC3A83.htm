Fecha / Hora del evento
Selector de fecha y hora
Muestra el atributo "fechaHora" del historico. Por defecto se inicializa con la fecha y hora del sistema.
Descripción
Textfield
Anexo
Selector de documento
 Muestra un documento asociado al evento y permite seleccionar otro o eliminar el asociado.
Si hay un fichero asociado, se muestra en forma de enlace (link) para su descarga, indicando entre parentesis su tamaño.
Admite formato PDF, con un tamaño máximo de 10 MB.
Dispone de un botón para seleccionar un fichero local (habilitado si no se ha especificado ningún fichero), y otro botón para eliminar el fichero previamente cargado (habilitado si ya se ha especificado un fichero).

Eliminar documento justificativo
Se llama al servicio REST de la entidad Documento, con la operación DELETE. Se elimina la asociación con el documento y se muestra "(ningún fichero seleccionado)".

Guardar
 Si se ha modificado el documento justificativo:
- Se carga el nuevo fichero en el servidor, haciendo "submit" del formulario con el selector del documento, llamando a la API "CargarFicheroDocumento". Mientras se envía el formulario  se muestra un indicador de carga. La API devolverá el identificador del nuevo documento, que se asocia al atributo "anexo" del historico.
