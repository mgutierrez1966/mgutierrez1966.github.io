 DESCRIPCION DEL PROCESO A EJECUTAR
Cambio de valor
Fichero con recursos
Se verifica que el fichero tiene extensión CSV y no supera 10 MB de tamaño. Si es así se muestra el error al usuario.
Clic
Cargar fichero
 Se carga el fichero en el servidor mediante la API CargarFicheroDocumento.
Si es correcto, se llama a la API ExpedienteAdquisicionApi.CargaRecursosPorFichero(), pasando el expediente de adquisición y el documento devuelto por la llamada anterior.
Se muestra al usuario un indicador de que el proceso se está ejecutando, bloqueando el resto de acciones.
Cuando la llamada devuelve la respuesta, se informa de ello al usuario con un mensaje modal.
Si el resultado ha sido correcto, tras cerrar el mensaje modal, se cierra la vista, devolviendo el resultado de la carga en la configuración de salida.
Si el resultado ha sido erróneo, se mantiene la vista abierta.
