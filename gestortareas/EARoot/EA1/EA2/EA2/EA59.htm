<b>Precondiciones:</b><br/>- Usuario identificado <br/>- El usuario es el propietario del tablero<br/><br/><b>Escenario principal:</b><br/>- El usuario selecciona la opción "Añadir lista"<br/>- El sistema solicita al usuario:<br/> - Nombre de la lista <br/> - Color de los elementos de la lista. Por defecto blanco.<br/> - Tipo de lista: normal (por defecto), borrador o finalizado<br/>- El sistema crea la lista.<br/> - Si ya existe una lista con el nombre indicado se notifica al usuario<br/> - Si ya existe una lista de tipo Borrador o Finalizado, y se ha especificado dicho tipo, se notifica al usuario que solo puede haber una lista del tipo Borrador o Finalizado en un tablero.<br/> - Las listas de tipo "Borrador" tiene el orden 0 (aparecen a la izquierda del tablero, como primera lista).<br/> - Las listas de tipo "Finalizada" tienen el orden 999 (aparecen a la derecha del tablero, como última lista).<br/>- El sistema actualiza la vista del tablero para mostrar la nueva lista.<br/><br/><br/><b>Postcondiciones:</b><br/>- Se ha creado la nueva lista.<br/><br/>
