 DESCRIPCION DEL PROCESO A EJECUTAR
Iniciar vista
Se cargarán las listas del tablero.
Se obtendrá del REST de Lista, el atributo "origenRestringido", que indica si el origen permitido es "Cualquiera" o solo determinadas listas.
En el caso de que el atributo "origenRestringido" sea TRUE, se usará el REST de MovimientoListaPermitido. Esta entidad almacena los pares "lista-origen / lista-destino" permitidos. Si se filtra por "destino" = <lista a modificar>, se obtendran en el atributo "origen" las listas origen permitidas.

Finalmente se marcarán los checkbox de las listas origen permitidas.
 
Clic
Guardar
 Se llamará ala API TableroAPI/permitirOrigenLista, pasando la lista a restringir, el parámetro "origenRestringido" a TRUE o FALSE en función de la opción seleccionada, y un array de identificadores de las listas origen marcadas (en su caso).
