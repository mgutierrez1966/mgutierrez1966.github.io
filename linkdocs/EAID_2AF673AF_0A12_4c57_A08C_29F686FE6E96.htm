PROCESOS Y ELEMENTOS FUNCIONALES

 
TIPO
NOMBRE
 DESCRIPCION DEL PROCESO A EJECUTAR
Inicializar
Vista
 Se obtienen los datos del Usuario cuyo UID se ha recibido como entrada, a través del servicio REST de Usuario.
Se inicializan los grids de "Roles asignados" y "Datos del rol" con los datos recogidos en los servicios REST de SRolBasico y SAtributoRolBasico, filtrando por el usuario.
Lista de roles asignados
Se inicializa con los roles asignados al usuario. Usa el servicio REST de la entidad SRolBasico para añadir o quitar roles.
Solo se permite especificar el tipo de rol para los roles añadidos, no para los ya existentes (estos no pueden cambiar su tipo).
La creación y eliminación de un rol conlleva la creación y eliminación en cascada de los atributos específicos del rol, por lo que ambas operaciones deberán llamar sincronamente al servicio REST y no podrán ser canceladas.
Datos del rol seleccionado
Se actualiza al seleccionar un rol, con el tipo de formulario específico del rol seleccionado.
Para los roles CISPOC y MANDO_ORGANICO, el campo "UCO apoyada" o "Mando Organico asociado" actualiza los 2 atributos asociados al rol, tanto el que contiene el ID como el que contiene el nombre de la UCO o Mando.
Restricciones por rol:
- Para el rol CISPOC, la UCO apoyada debe ser una entidad DependenciaET.
- Para el rol MANDO_ORGANICO, el Mando asociado debe ser una entidad de tipo MandoEt.

** INICIO ARIGIS-89
- Roles SEAPO, ADMINISTRADOR y DIRECTOR: No se muestra ningún atributo del rol.

** INICIO ARIGIS-272
- Rol MANDO ORGANICO: Se muestra el MandoEt asociado. Se permite seleccionar un nodo de tipo MandoEt o Bt mediante el formulario FormSeleccionUCO.
** FIN ARIGIS-272

- Rol CISPOC: Se muestra la DependenciaEt asociada. Se permite seleccionar un nodo de tipo AlmacenEt DependenciaEt o subclases (BT, CECOM), mediante el formulario FormSeleccionUCO.
- Rol SUBDIRECTOR: Se muestra el BTasociado. Se permite seleccionar un nodo de tipo BT mediante el formulario FormSeleccionUCO. Como nodo raiz del formulario de selección de UCO se establece el RT-22.
- Rol OACISE: Se muestra el CECOM asociado. Se permite seleccionar un nodo de tipo CECOM mediante el formulario FormSeleccionUCO. Como nodo raiz del formulario de selección de UCO se establece el RT-22.
** FIN ARIGIS-89
