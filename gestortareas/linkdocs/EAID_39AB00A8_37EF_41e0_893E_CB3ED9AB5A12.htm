 DESCRIPCION DEL PROCESO A EJECUTAR
Clic
Generar (Resumen de tareas de grupo)
 Se determinan los grupos de los que es miembro el usuario, mediante el REST de MiembroGrupo (filtrando por el id de miembro, y obteniendo los distintos grupos).

Si el usuario pertenece a más de un grupo, se muestra la vista FormSeleccionGrupoInforme,pasando la lista de los grupos obtenidos,para la seleccion del grupo para el que desea generar el informe.

Una vez se tenga el id del grupo para el que se desea el informe, se llama a la API InformesApi/informeTareasGrupo, pasando el identificador de grupo. La API devuelve un fichero PDF con el informe generado.

Generar (Estado de tableros)
 Se llama a la API InformesApi/informeEstadoTableros. La API devuelve un fichero PDF con el informe generado.
