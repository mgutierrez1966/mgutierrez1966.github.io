Bloque "Recursos para baja"
Tab
 Muestra los recursos HW (entidad RecursoHw) incluidos en el expediente de baja.
Obtiene los datos del servicio REST RecursoHw, filtrando por el atributo expedienteBajaHw.
En el título muestra el total de recursos en la lista.
La tabla permite SELECCION MULTIPLE.
** INICIO ARIGIS-287

** FIN ARIGIS-287
 
Opciones del bloque "Recursos para baja"
Toolbar
  • Añadir recursos: Habilitado solo si el estado del expediente es BORRADOR o CLASIFICACION..
  • Quitar recursos: Habilitado en los mismos casos que "Añadir recursos", y además si hay seleccionado uno y un solo recurso en la lista.
  • Ver recurso: Habilitado si hay seleccionado uno y solo un recurso en la lista.
  • Clasificar recursos: Visible si el estado del expediente es CLASIFICACION. Habilitado si la lista de recursos para baja no está vacía.
  • Exportar lista: Habilitado siempre
Bloque "Documentos"
Tab
 Muestra los documentos (entidad Documento) asociados al expediente.
En el título muestra el total de documentos en la lista.
Opciones del bloque "Documentos"
Toolbar
  • Añadir documento: Habilitado siempre
  • Borrar documento: Habilitado si hay algún documento seleccionado
  • Abrir documento: Habilitado si hay algún documento seleccionado.
Lista de Documentos
Grid
 Muestra la lista de documentos (entidad Documento) asociados al expediente.

PROCESOS Y ELEMENTOS FUNCIONALES

 
TIPO
NOMBRE
 DESCRIPCION DEL PROCESO A EJECUTAR
Mostrar vista
Inicializa los datos de la pestaña "Datos básicos".
Obtiene el total de los recursos para baja y los documentos del expediente, para mostrar el total en los títulos de las pestañas.
Clic
Modificar
Muestra  la vista FormExpedienteBajaHw con los datos del expediente. Si la edición del elemento es exitosa, actualiza el modelo.
Clic
Anular
 Solicita confirmación para anular el expediente de baja.
Llama a la operación AnularExpedienteBajaHw del servicio API ExpedienteBajaHwAPI. Si la llamada es exitosa, se recarga la vista completa para actualizar la información y el estado de los componentes, conforme al nuevo estado del expediente.
Clic
Borrar
 Solicita confirmación para eliminar permanentemente el expediente del sistema.
Elimina el objeto mediante el servicio REST de ExpedienteBajaHw.
Si la llamada al servicio es correcta, cierra la vista y actualiza la lista de expedientes de baja de la Vista General.
Clic
Cambiar a CLASIFICACION
 Solicita confirmación para cambiar el estado del expediente de baja a "Clasificación".
Llama a la operación CambiarExpedienteBajaHwClasificacion del servicio API ExpedienteBajaHwAPI, pasando el identificador del expediente.
Si la operación es correcta, recarga los datos del expediente y actualiza la vista conforme al nuevo estado.
Clic
Solicitar APROBACION
 Solicita confirmación para cambiar el estado del expediente de baja a "Pendiente aprobación".
Llama a la operación SolicitarAprobacionExpedienteBajaHw del servicio API ExpedienteBajaHwAPI, pasando el identificador del expediente.
Si la operación es correcta, recarga los datos del expediente y actualiza la vista conforme al nuevo estado.
Clic
Aprobar propuesta clasificación
 Solicita confirmación para cambiar el estado del expediente de baja a "Aprobado".
Llama a la operación AprobarExpedienteBajaHw del servicio API ExpedienteBajaHwAPI, pasando el identificador del expediente.
Si la operación es correcta, recarga los datos del expediente y actualiza la vista conforme al nuevo estado.
Clic
Finalizar
 Solicita confirmación para finalizar el expediente de baja HW.
Se muestra la vista FormFechaCierreExpedienteBajaHw, para solicitar la fecha de cierre del expediente de baja.
Si la vista se ejecuta con éxito, se llama a la operación FinalizarExpedienteBajaHw del servicio API ExpedienteBajaHwAPI, pasando el identificador del expediente y la fecha de cierre especificada.
Si la operación es correcta, recarga los datos del expediente y actualiza la vista conforme al nuevo estado.
Clic
Listados oficiales
 ** INICIO ARIGIS-98
Muestra la vista FormOpcionesListadoExpBaja , pasando el expediente de baja actual.
** FIN ARIGIS-98
Clic
Añadir recursos
 Muestra la vista "FormSeleccionRecursosHwBaja".
Si la vista tiene exito, llama a la operación AgregarRecursosExpedienteBajaHw de la API ExpedienteBajaHwAPI, pasando la lista de recursos seleccionados.
Si la llamada a la API tiene exito, se actualiza el grid de recursos para baja, para reflejar los cambios.
 
Clic
Quitar recursos
 Solicita confirmación para quitar del expediente de baja los recursos seleccionados.
Llama a la operación QuitarRecursosExpedienteBajaHw de la API ExpedienteBajaHwAPI, pasando los recursos seleccionados. Si la llamada a la API tiene exito, se actualiza el grid de recursos para baja, para reflejar los cambios.
Clic
Ver recurso
Abre en una nueva pestaña la vista PnRecursoHWDetalle con los datos del recurso seleccionado. La nueva pestaña puede cerrarse.
Clic
Clasificar recursos
 Muestra la vista "FormClasificacionRecursosHwBaja", pasando el identificador del expediente de baja.
Si la vista tiene exito, se actualiza el grid de recursos para baja, para reflejar los cambios.
Clic
Exportar lista
Comportamiento estandar
Cambio de valor
Cuadro de búsqueda libre de recursos
 ** INICIO ARIGIS-287
Modifica el parámetro "query" de la llamada al servicio REST asociado al store de recursos para baja.
** FIN ARIGIS-287
Clic
Añadir documento
 Muestra el explorador de ficheros del sistema para seleccionar un documento.
Si el fichero no tiene formato PDF o supera los 10 MB, se mostrará un error informando de dichas limitaciones.
Se llama al servicio API CargarFicheroDocumento y pasando los datos del fichero, recuperando el identificador del nuevo objeto Documento.
Si la operación es correcta, se llama a la operación AgregarDocumentoExpedienteBajaHw de la API ExpedienteBajaHwAPI, pasando el identificador del nuevo documento.
Se actualiza la lista de documentos.
Clic
Borrar documento
 Se solicita confirmación para eliminar el documento seleccionado.
Se llama a la operación BorrarDocumentoExpedienteBajaHw de la API ExpedienteBajaHwAPI, pasando el identificador del documento.
Se actualiza la lista de documentos.
Clic
Abrir documento
Se llama al servicio API LeerFicheroDocumento y el ID del documento seleccionado, y el resultado se mostrará en una nueva ventana del navegador.
